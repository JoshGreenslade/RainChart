**To complete a challenge:**
1. Read the issue carefully
2. Do the work (code, write, plan, present)
3. Post your response as a comment
4. Close the issue when done
5. Receive automated grading and feedback

### Step 4: Track Your Progress

Weekly progress updates show:
- Current level and XP
- Badges earned
- Module completion
- Streak status
- Recommended next steps
- Leaderboard position (if multiple trainees)

### Step 5: Boss Battles (Level 5+)

Once you reach Level 5, unlock monthly boss battles:
- Epic 3-5 day challenges
- Combine all skills
- Massive XP rewards
- Special badges

## Troubleshooting

### "I'm not getting daily challenges"
- Check workflows are enabled in repository settings
- Verify your profile exists in `.github/context/`
- Check `assets/training-data` branch for progress file
- Try `workflow_dispatch` to trigger manually

### "Grading seems incorrect"
- Post feedback as issue comment
- AI will reconsider and adjust
- Learning curve for AI grading quality

### "Too easy / too hard"
- Scenarios scale to your level
- Request specific difficulty
- Skip ahead if too easy
- Ask for hints if stuck

### "Not enough time"
- Focus on daily challenges only
- Skip optional content
- Boss battles are optional
- Quality > quantity
